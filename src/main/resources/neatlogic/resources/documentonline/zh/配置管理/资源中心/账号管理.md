# 账号管理
账号管理页面是用来配置连接协议和账号的页面，先添加连接协议，才能添加账号，账号需要关联协议。

## 协议管理
协议管理是用来管理协议的列表，协议需填写的信息有协议名和默认端口号，支持添加、编辑、删除协议。
![](images/账号管理_添加协议.gif)

## 账号
账号是应用于资产配置，支持连接远程工具使用，所以添加账号时，连接协议、账号名、密码必须填写。
