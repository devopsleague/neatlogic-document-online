# 应用清单
应用清单是按应用模块汇总资产，并回显资产巡检结果。

## 资产清单
资产清单列表中回显当前选中的应用或模块包含的资产对象及其巡检结果。
<br>
访问权限：用户需拥有资源中心-清单权限
![](images/应用清单_资产清单.png)
点击资产的IP地址，可查看资产详细信息
![](images/应用清单_查看资产详情.gif)
